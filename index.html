Python + TensorFlow (local only)

HDR → Tone‑mapped PNG

The Python script decodes assets/HDR_041_Path_Ref.hdr with TensorFlow, tone‑maps it, and saves generated/hdr.png. This page simply displays that PNG.

Run: python3 serve.py Open: http://localhost:8000

How to run

  1. Install TensorFlow CPU in your Python 3.12 environment: pip install tensorflow-cpu.
  2. From the repo root, run python3 serve.py.
  3. Open http://localhost:8000 in a browser; the script renders generated/hdr.png on start.

If the image does not appear, check the terminal for TensorFlow or file errors. The server rebuilds the PNG each time it starts.
